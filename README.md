# K-Beauty 라오스 쇼핑몰 (UI 프로토타입)

한국 ↔ 라오스 양국간 뷰티 직거래 쇼핑몰의 화면 구성 프로토타입입니다.

## 구성된 화면

1. **홈** (`/`) — 배너, 카테고리, 베스트셀러, 신상품
2. **상품 상세** (`/product/[id]`) — 이미지, 옵션, 리뷰, 탭 정보
3. **장바구니** (`/cart`) — 상품 선택/수량/삭제, 합계, 무료배송 안내
4. **결제** (`/checkout`) — 라오스 배송지, 결제수단(카드/계좌이체/간편결제), 주문완료

## 기능

- 한국어 ↔ 라오스어 실시간 전환 (헤더 토글)
- 한국 원화(KRW) ↔ 라오스 킵(LAK) 자동 환산 표시
- 데스크톱/모바일 반응형
- 장바구니 상태 zustand로 유지 (localStorage)

## 실행 방법

```bash
npm install
npm run dev
```

http://localhost:3000 에서 확인하세요.

## 향후 추가 예정

- 로그인 / 회원가입
- 마이페이지 / 주문조회 / 배송 추적
- 판매자 관리자 화면
- 실제 결제 연동 (KakaoPay, BCEL One)
- 데이터베이스 연동 (상품, 주문, 회원)
